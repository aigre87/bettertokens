Application for TSA Qualified Issuer status
Projects that comply with BetterTokens standards obtain Qualified Issuer status, which guarantees membership in BetterTokens, verified token status on Waves, a ticker, a logo and a priority search listing on Waves DEX.
Pre-application

Potential applicants are encouraged to study documents on the BetterTokens website, namely the application process and application form, Code of Conduct and Detailed Description of Standards. Any questions or requests for clarification should be addressed to sec@bettertokens.org before starting the application process.

1
Transfer Application Fee

The applicant transfers a non-refundable Membership Application Fee of 1,000 WAVES to wallet address 3PAyqLbvEwQj6Y2fgKUc6u3DGxMxVaKbRQL.

This fee covers processing of the application and does not guarantee that the applicant will obtain TSA Qualified Issuer Status.

2
Submission of application via email

The applicant submits the documents listed below using the application form available on the BetterTokens website.

The applicant must attach the following documents to the application form, adding relevant numeric indicators, provided in square brackets below, to the file names.

All documents shall be submitted in English in a readable format.

List of documents
  • [000] Logo requested for Waves client (svg format only, 20kB)
  • [001] Legal information concerning tokenisation project (Certificate of Incorporation; Articles; Charter)
  • [002] White Paper
  • [003] Detailed technical specifications for the project
  • [004] Token sale agreement (or equivalent) / Private sale terms (if applicable) / Bounty programme terms (if applicable)
  • [005] Website and Marketing materials
  • [006] AML / KYC Policy or measures description
  • [007] Internal risk management policy or measures description
  • [008] List of key relevant markets and investors targeted by tokenisation project
  • [009] Results of independent security audit (if conducted) / internal code review (if conducted)
  • [010] Legal opinion (if obtained)
  • [011] Any other documents considered by the applicant as having crucial importance for the application process.
3
Receive an automatic confirmation

Once the application is submitted, the applicant will receive an automatic reply stating that BetterTokens will review the provided documents and decide on the applicability of Qualified Issuer Status to the project.

The total timeframe for this process is 7 business days.

Review
1
Compliance with the application process

BetterTokens will review the application documents for compliance with the application process indicated on its website.

If the application does not meet the necessary requirements, it will be discarded with no further consideration, with corresponding notice sent to the applicant.

Applications submitted in compliance with the application process will proceed to the next stage.

2
Scoring

BetterTokens will review the application documents for compliance with its tokenisation standards.

Special attention will be given to the information presented by the project to its investors, including its white paper, AML / KYC standards and clarity of disclosures regarding the token and secondary market trading.

The review will be undertaken following proprietary scoring methodology.

Awarding Qualified Issuer Status

Scoring results for applicants will be presented to BetterTokens Board members, who will formalise the award of Qualified Issuer status.
